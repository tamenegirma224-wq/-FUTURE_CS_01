## 🔗 Project Title
Dormitory Management System API Security Risk Analysis

##  Description
This project is a Dormitory Management System with security analysis features. 
The system manages student dormitory information, room allocation, and admin access securely.

##  Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- XAMPP
- MySQL Database

##  System Features
- Student registration
- Room management
- Admin login system
- Database storage using MySQL
- API security analysis

## Security Features
- Admin password encryption
- Secure login authentication
- Database protection
- Input validation

##  Database
The system uses XAMPP MySQL database for storing:
- Student information
- Room data
- Login credentials

Admin passwords are encrypted before storing in the database to improve security.
